# Network-mapper
Builds a network map by pressing a single button

Этот Python-скрипт реализует веб-приложение для анализа и визуализации устройств в локальной сети. Он использует Flask для создания веб-сервера и несколько сторонних библиотек (`psutil`, `python-nmap`, `socket`, `ipaddress` и другие).

### Назначение:
Программа сканирует все активные подсети текущего компьютера, определяет устройства в этих подсетях (их IP, MAC-адрес, имя хоста, производителя и тип), и отображает результаты в виде интерактивной таблицы и графа (сетевой диаграммы) на веб-странице.

### Ключевые особенности:
- **Визуализация**: Используется библиотека `vis-network` для отображения топологии сети, где шлюзы — центральные узлы, а подключенные устройства — периферийные.
- **Определение типа устройства**: По MAC-адресу и имени хоста скрипт пытается определить, является ли устройство принтером, телефоном, телевизором, маршрутизатором и т.д.
- **Проверка прав администратора**: На Windows система проверяет, запущен ли скрипт от имени администратора. Это важно для получения MAC-адресов и корректной работы `nmap`.
- **Потоковая передача данных (SSE)**: Результаты сканирования передаются на клиентскую часть по мере их получения, что позволяет отображать прогресс и логи в реальном времени.

### Необходимые зависимости:
Для корректной работы скрипта необходимо:
1. Установить Python.
2. Установить требуемые пакеты:
   ```bash
   pip install flask psutil python-nmap
   ```
3. Установить `nmap` на операционную систему и добавить его в переменную окружения `PATH`.

### Как использовать:
1. Запустить скрипт от имени администратора.
2. Открыть в браузере `http://127.0.0.1:5000`.
3. Нажать кнопку "СКАНИРОВАТЬ СЕТЬ".
4. Наблюдать за процессом в логах и получить итоговую таблицу и визуализацию.

Этот инструмент полезен для диагностики сети, выявления несанкционированных подключений и понимания структуры домашней или офисной сети.
